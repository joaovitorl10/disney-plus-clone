Bem-vindo ao Disney Clone

Este projeto foi desenvolvido como uma demonstração de como criar uma plataforma de streaming inspirada na Disney usando Python e Flask.

Tecnologias Utilizadas

  • Python - Linguagem principal
  • Flask - Framework web
  • HTML5 - Estrutura das páginas
  • CSS3 - Estilização
  • JavaScript - Interatividade
  • Bootstrap - Design responsivo

Funcionalidades

Catálogo de Filmes

Navegue por uma coleção de filmes Disney

Trailers Integrados

Assista aos trailers diretamente na plataforma

Busca e Filtros

Encontre filmes por título, ano ou avaliação

Design Responsivo

Funciona perfeitamente em todos os dispositivos

Como Executar

1. Clone o repositório:
git clone [url-do-repositorio]
2. Instale as dependências:
pip install -r requirements.txt
3. Execute a aplicação:
python app.py
4. Acesse no navegador:
http://localhost:5000
